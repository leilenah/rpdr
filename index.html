You're a Winner, Baby Random Variable

Notation: $X \sim Win(d, r, a, g)$
Description: A boolean variable that is 1 with probability $p$, which is dynamically generated using $d,r,a,g$.
Parameters: $d$ - number of episodes.
$r$ - number of episodes contestant has won.
$a$ - number of episodes contestant placed bottom two.
$g$ - number of contestants remaining.
